## Feature Requirements

1. Bounce Sprite off walls.
1. Velocity controls:
    - Create new property on Ball constructor
    - 'speed' feature: keys 1-9 will control velocity
1. Flexible controls:
    - Z key slows
    - X key speeds
    - C key enlarge ball
    - V key shrink ball
    - Possible bugs when exceeding neg zero
